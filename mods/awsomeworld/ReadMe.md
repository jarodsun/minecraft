# AwsomeWrold

## 2023年10月19日

### 添加了`9`个矿物维度

1. 煤炭世界
2. 铁世界
3. 金世界
4. 钻石世界
5. 远古残骸世界
6. 绿宝石世界
7. 青金石世界
8. 红石世界
9. 石英世界

每个维度都是1层基岩，60层矿石，1层草方块。

每个维度的传送台，使用“矿物”+“讲台”进行合成。

每个维度光照强度都是15，不会产生敌对生物。

没有村庄。

### 添加了`1`个和平维度

1层基岩，60层石头，1层草方块。

传送台，使用“石头”+“讲台”进行合成。

光照强度始终是15，不会产生敌对生物。

有村庄。可以刷村民。很适合建造基地。
